Git

1. Introduction

git is a version control system which can be used for collabrating into projects together and book keeping of all the changes we made in the repository.

2. Initialize git 

 git init
It is used to intitalize git software package
Example output 
 Initialized empty Git repository in /home/bthakar/Desktop/demoC/.git/

3. Configure git 

 git config --global user.name "#gitusername"
Add user name 
 git config --global user.email "#gitemail"
decription : Add email id
We can use three flags --global --local --system. global for declaring the config file globally on computer; local for declaring the config file for a particular directory ; system for declaring the config file for particular user

4. Add remote origin 

 git remote add origin </link>
It is used to add remote repository link

5. Add files to staging index 

 git add .
It is used to add files to staging index

6. Commit files to repository 

 git commit -m "Initial Commit"
It is used to commit the changes from staging index
Example output 
 [master (root-commit) 18116c6] Initial Commit
 1 file changed, 2 insertions(+)
 create mode 100644 hello.txt

7. Push to repository 

 git push origin master
It is used to push the committed changes to the remote repository
